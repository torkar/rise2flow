# Rise 2 Flow replication package

In `docs/` one can find the R Markdown file (`index.Rmd`) which, when executed in `RStudio` compiles to `index.html`. In `data/` all data for the analysis can be found in `csv` format.

A good start would be to clone the directory and open the project in `RStudio` by clicking `rise2flow.Rproj`.
